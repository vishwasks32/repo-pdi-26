# Next App v1 (Next.js)

A starter Next.js App Router application demonstrating basic page rendering and routing.

## Features

- Next.js App Router
- Static home page
- Example folder structure for scaling with nested routes

## Folder Structure

- `app/page.js` - home page
- `app/about/page.js` - about page
- `app/dashboard/page.js` - dashboard route
- `app/listings/[id]/page.js` - dynamic route example
- `app/components/*` - shared components

## Prerequisites

- Node.js 18+
- npm / yarn / pnpm

## Setup & Run

```bash
cd Fronten-apps/nextappv1
npm install
npm run dev
```

Visit `http://localhost:3000`.

## Notes

- Update `next.config.mjs` or `app/globals.css` for project-specific assets or styling.
