# Grade Calculator

A C# console application that calculates a student's GPA based on their courses and grades.

## Description

This application demonstrates object-oriented programming by creating Student and Course classes. It calculates the GPA by weighting grades with credit hours.

## Features

- Student class with course management
- Course class with grade and credit hour properties
- GPA calculation algorithm
- Sample data demonstration

## Prerequisites

- .NET 9.0 SDK

## How to Run

1. Navigate to the project directory:
   ```bash
   cd grade_calculator
   ```

2. Run the application:
   ```bash
   dotnet run
   ```

## Sample Output

```
Student: Vishwas Singh
GPA: 3.67
```

## Classes

- **Course**: Represents a course with name, grade, and credit hours
- **Student**: Manages a list of courses and calculates GPA
