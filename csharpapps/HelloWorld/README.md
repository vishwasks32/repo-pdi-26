# HelloWorld

A simple C# console application that demonstrates basic object-oriented programming concepts.

## Description

This application creates an instance of a class from a custom namespace and calls a method to print "Hello World from MyNamespace" to the console.

## Features

- Demonstrates namespace usage
- Shows basic class and method implementation
- Simple console output

## Prerequisites

- .NET 9.0 SDK

## How to Run

1. Navigate to the project directory:
   ```bash
   cd HelloWorld
   ```

2. Run the application:
   ```bash
   dotnet run
   ```

## Output

```
Hello World from MyNamespace
```
