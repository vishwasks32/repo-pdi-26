# Username Extractor

A simple C# console application that extracts the username from an email address.

## Description

This application takes an email address as input and extracts the username part (before the @ symbol). It includes basic validation to ensure the input contains an @ symbol.

## Features

- Email validation
- Username extraction
- Error handling for invalid emails

## Prerequisites

- .NET 9.0 SDK

## How to Run

1. Navigate to the project directory:
   ```bash
   cd usernameExtracter
   ```

2. Run the application:
   ```bash
   dotnet run
   ```

## Sample Output

```
The username is: vishwas
```

## Usage

The application uses a hardcoded email address for demonstration. In a real-world scenario, you would modify it to accept user input.
